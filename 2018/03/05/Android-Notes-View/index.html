Android 绘图机制处理

  • 系统屏幕密度，密度值，分辨率对应关系

  • 独立像素密度(密度无关像素 (dp)在定义 UI 布局时应使用的虚拟像素单位，用于以密度无关方式表示布局维度 或位置)

  • 单位转换

  • 2D绘图

    • setAntiAlias()//画笔锯齿
    • setColor()//画笔颜色
    • setARGB()
    • setAlpha()
    • setTextSize()
    • setStyle()//空心，实心
    • setStrokerWidth()//边框宽度

  • XML

    • Bitmap

    • Shape

    • Layer

    • Selector

  • 绘图技巧

    • Canvas

    • Board

  • layer图层

  • 色彩特效

    • 图片的数据结构常使用位图Bitmap,由点阵(像素点矩阵)和颜色值(ARGB)组成
    • 色彩处理中包含色调，饱和度，亮度，使用ColorMatrix(4*5颜色矩阵)

    • 滑动seekbar，改变颜色混合效果

    • Android不允许直接修改原图，创建同大小的位图，并将原图绘制到该Bitmap

  • ColorMatrix

    • 图像反转

  • 获取图像像素值,三种像素点处理

  • 图像变换矩阵初始矩阵

  • 图像处理基本变换

    • Translate
    • Rotate 以坐标原点为中心旋转
    • Scale 每个点的坐标等比例缩放
    • Skew 保持所有点的x或y轴坐标不变，对应的y或x坐标等比例平移，有水平错切和垂直错切

  • drawBitmapMesh()像素块

  • 旗帜FlagView
